CSS First Steps

Simple selectors

Selects elements based on name, id, class.

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page.

Element Selector

To use it, just write the element name. The element selector selects all html elements with the specified name.

Id Selector

The id selector uses the id attribute of an html element to select it. To use it, write a hash (#) character, followed by the id of the element.

The id of an element is unique within a page, so the id selector is used to select one unique element by page.

Class Selector

The class selector selects all the html elements with a specific class attribute. To use it, write a period (.) character, followed by the class name.

Note: Html elements can also refer to more than one class by adding a list separated by spaces.

Class and Name Selector

The class and name selector selects all the html elements with both a specific class attribute and a specific name. To use it, write the name followed by a period (.) character, followed by the class name.

The CSS Grouping Selector

The grouping selector selects all the html elements whose name is specified. To use it, write a sequence of element names separated by a comma (,).
