CSS First Steps

How to applied the CSS code to an html document

Before start seeing the specific properties by element and the different selector types, we'll see the 3 ways to applied the css code to an html document and where does it has to be inserted in each one, in addition to how does the property overrides works.

The 3 Ways to insert CSS code

CSS inline

In this way we only can use the declaration block's content syntax to create a single style rule that only applies to one single element.
To use inline styles, add the style attribute to the relevant element. The style attribute value can be any sequence of CSS declarations.

CSS internal

An internal style sheet only styles one single HTML page with any number of style rule.
The internal style rules are defined inside a <style> element, inside the head section.

CSS external

With an external style sheet, we can change the look of any html by changing just one file.
Each html page must include a reference to the external style sheet file inside the <link> element, inside the head section. We can do this by adding in that element the "href" attribute with a value of the .css file's URL and the "rel" attribute with the value: "stylesheet".

Priority order of CSS code sources of different types

When there are multiple assignments for the same property from css codes of different types (inline/internal/external), the one that has the highest priority will override all the others.

The priority order is the next, where the first has the lowest priority:

  1. External and internal style sheets
  2. Inline style
In the case of the style sheets, the order followed corresponds to the order of the <link> and <style> tags in the html document.

This means that the styles in a page will "cascade" into a new "virtual" style sheet that corresponds to the screen shown result.
