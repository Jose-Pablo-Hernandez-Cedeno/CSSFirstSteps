CSS First Steps

Inline CSS, Internal CSS and External CSS

CSS inline

CSS inline is the way we call the CSS code that style an html document by adding the style attribute of the relevant html elements with a value that can be any sequence of CSS declarations separated by semicolons (";") where, optionally, the last can end with semicolons.

When styling this way, we can't reuse the assignements added to a particular element.

CSS internal

CSS internal is the way we call the CSS code that comes from an internal style sheet.

The internal style sheets are the sets of any number of style rules that are defined inside a <style> element, inside the head section. Those only can style the page where they was declaraded but allow the reuse of assignments in different elements by selectors.

CSS external

CSS external is the way we call the CSS code that comes from an external style sheet.

The external style shets are .css files that contains any number of style rules. With this kind of style shets, we can change the look of any html document by changing just one file.

Each html page that has to be styled with external CSS must include a reference to the external style sheet file inside the <link> element, inside the head section. We can do this by adding in that element the "href" attribute with a value of the .css file URL and the "rel" attribute with the value: "stylesheet".
