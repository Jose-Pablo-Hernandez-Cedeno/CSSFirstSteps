CSS First Steps

background-image

Description

This property allows to set one or more images to use as the element background.

By default, a background-image is placed at the top-left corner of an element (see background-position to change that), and repeated both vertically and horizontally so it covers the entire element background (see background-repeat to change that).

Values

Value Description
url("URL") The URL of the image. To specify more than one image, write multiple values by following this syntax for each one and separate them with a comma.
none No background image will be displayed. This is default.
