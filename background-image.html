CSS First Steps

background-image

Description

The background-image property can be used to set one or more background images for an element.

By default, a background-image is placed at the top-left corner of an element (see background-position to change that), and repeated both vertically and horizontally so it covers the entire element background (see background-repeat to change that).

Values

Value Description
Color values It can takes any valid CSS Color value.
Transparent Specifies that the background color should be transparent. This is default.
