CSS First Steps

Combinators

Descendant combinator

Selects the elements that are descendants of some first selector matching element and match the second selector at the same time.

Syntax:
-one or more CSS white space characters-
(tab/space/carriage return/form feed/new line)

Child combinator

Selects the elements that are direct children of some first selector matching element and match the second selector at the same time.

Syntax:
>

General sibling combinator

Selects the elements that have the same parent as, and follow, though not necessarily immediately, some first selector matching element and match the second selector at the same time.

Syntax:
~

Adjacent sibling combinator

Selects the elements that immediately follow some first selector matching element and match the second selector at the same time.

Syntax:
+
