Property overrides

When there is more than one css code inserted to a html document specifing different values for the same property, all the assignements to the property are processed in a specific order and the last finded will override the previous.

The mentioned order followed is the next:

  1. Browser default
  2. External and internal style sheets
  3. Inline style
In the case of the style sheets, the order follows corresponds to the order of the <link> and <style> tags in the html document.

This means that the styles in a page will "cascade" into a new "virtual" style sheet that corresponds to the screen shown result.
