CSS First Steps

Basic Selectors

Universal Selector

The CSS universal selector (*) matches all the elements.

Syntax:
*

Type selector

Selects all elements that have the given node name.

Syntax:
elementname

Class selector

Selects all elements that have either a class list that includes the given class or one class value equals to it.

Syntax:
.class_name

ID selector

Selects all elements that have the given node name.

Syntax:
#id_value
