Attribute selectors

Case sensitivity flags

All attribute selectors, except for the "[attr]", follow the next syntax:

Syntax:
[attr operator value flag]

where flag is optional and can be either i(or I) or s(or S). When the first is present, the attribute values will be comparated case-insensitively, and, when the second is present, those will be comparated case-sensitively.

This fact only applies for attribute values with characters within the ASCII range and case-sensitive matching via s flags is only possible in systems that preserve the original case, so it has less browser compatibility.

The Selectors

[attr=value]

[attr~=value]

[attr|=value]

[attr^=value]

[attr$=value]

[attr*=value]
