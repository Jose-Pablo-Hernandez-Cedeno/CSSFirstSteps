background-position

Description

This property allows to set the starting possition of the background image.

Values

Value Description
xpos ypos The first value is the horizontal position and the second value is the vertical. The top left corner is 0 0 in any unit used. Units for both can be pixels (0px 0px) or any other CSS units. If you only specify one value, the other value will be 50%. You can mix % and position
left top
left center
left bottom
right top
right center
right bottom
center top
center center
center botto 		If you only specify one keyword, the other value will be "center". left top is default.
