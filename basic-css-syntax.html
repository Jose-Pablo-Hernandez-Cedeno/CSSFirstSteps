CSS Properties

The CSS Properties are the predefined categories of element's concrete visual characteristics/atributtes that CSS provides to allows the programmers to set a member of each category as an actual visual characteristic of the element styled. For that purpose, the lenguaje provides to each one a property name and one possible property value at time.

Porperty name

The property name is a predefined keyword that css bring us to identify and be able to refer to any particular property without possibility of conffusion. For that reason, every property have a unique and distinctive name.

Property value

The predefined element's concrete visual characteristics mentioned before are called the values of the CSS property, has one associated set of possibles values.
