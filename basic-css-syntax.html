CSS First Steps

Basic css syntax

CSS Comments

The CSS comments are not displayed in the browser, but they can help to document our source code. They starts with /*, ends with */ and can be sinlge line and multi line.

CSS Properties

The CSS Properties are a predefined disjoint categories of element's possible concrete visual characteristics/atributtes. CSS provides the properties to make possible to the programmers the set of a member of each category as an actual visual characteristic of the element styled. For that purpose, the lenguaje provides to each one a property name and a set of values wich can be used in a expression of the lenguaje called declaration.

Porperty name

The property name is a predefined keyword that css bring us to identify and be able to refer to any particular property without possibility of conffusion. For that reason, every property have a unique and distinctive name.

Values

A value in CSS is a predefined keyword that represents an element's particular possible concrete visual characteristic that can be asigned as an actual visual characteristic of the element styled by a declaration.

CSS Declaration

CSS declarations are the css basic units, these allows the programmers to assign a specific value to a specific property.
Declaration syntax structure consists of the CSS property name and a value to assign, separated by a colon (":"), representing the operation of assignement.

Style Rule

A CSS rule consists of a selector and a declaration block. These style rules allows the programers to applied the value assignations to the properties of the elements he wants

Selectors

CSS selectors are the part of the style rule that allows the programmer to indicate (or select) the HTML elements whose properties will be assign in the declarations

Declaration block

A declaration block is a sequence, that is surrounded by curly braces ("{}"), of one or more declarations separated by semicolons (";"). The last declaration can end with semicolons, and the space, tab and newline characters are ignored.
