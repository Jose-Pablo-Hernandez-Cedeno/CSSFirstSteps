CSS Properties

The CSS Properties are a predefined disjoint categories of element's possibles concrete visual characteristics/atributtes. CSS provides the properties to make possible to the programmers the set of a member of each category as an actual visual characteristic of the element styled. For that purpose, the lenguaje provides to each one a property name and one possible property value at time wich can be used in a expression of the lenguaje called Declaration.

Porperty name

The property name is a predefined keyword that css bring us to identify and be able to refer to any particular property without possibility of conffusion. For that reason, every property have a unique and distinctive name.

Property value

The predefined element's concrete visual characteristics mentioned before are called the values of the CSS property, has one associated set of possibles values.
